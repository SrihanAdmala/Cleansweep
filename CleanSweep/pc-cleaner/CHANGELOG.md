## [1.0.0] - 2025-01-01

### Added
- Initial release
- Junk cleaner for Windows, macOS, and Linux
- Browser cache cleaning (Chrome, Firefox, Edge)
- App cache cleaning (Discord, Teams, Spotify)
- System optimizer (DNS flush, IP refresh, Recycle Bin)
- Live system dashboard (CPU, RAM, Disk)
- Startup program viewer (Windows)
- Large file scanner (>50MB)
- One-click installer (`install.py`) with desktop shortcut creation
- Desktop launchers: `CleanSweep.bat` (Windows), `cleansweep.sh` (macOS/Linux)
- Real-time streaming clean log (Server-Sent Events)
- Dark industrial dashboard UI
- pip-installable package (`setup.py`, `pyproject.toml`)
- GitHub Actions automated release workflow
